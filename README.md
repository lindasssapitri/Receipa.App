# Recepia App

Recepia App adalah proyek aplikasi kuliner yang dikembangkan untuk mempermudah manajemen resep masakan. Aplikasi ini memungkinkan pengguna untuk menjelajahi berbagai resep dan mengorganisir instruksi memasak favorit mereka dengan rapi.

## 🚀 Fitur Utama
- **Pencarian Cerdas:** Temukan resep masakan untuk hari ini dengan cepat.
- **Perpustakaan Resep:** Lihat resep yang tersimpan dalam tampilan grid yang elegan.
- **Profil Pengguna:** Kelola data diri dan informasi akun dengan mudah.

## 🛠 Teknologi yang Digunakan
- **Framework:** Flutter
- **Bahasa:** Dart
- **Manajemen State:** Flutter native state management
- **UI/UX Design:** Figma
